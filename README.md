# i-spy
Play the I Spy game with the Kinova Gen3 Robot!

## Software Requirements:
- [Ubuntu 18.04.6]
- [ROS Melodic]
- [rospy]
- [darknet_ros]
- [OpenCV]
- [opencv-python] v4.2.0.32
- [SpeechRecognition]
- [pyttsx3]
- [cv_bridge]
- [smach]
- [actionlib]
- [sensor_msgs]
- [roslaunch]
